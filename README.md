## 🛠️ Managing your agents

Rig works with standard AI coding agents. You select your preferred agent within the settings menu. 

### Adding an agent
Select the Settings tab inside the app. Choose your agent provider. Enter your access key into the provided field. Rig saves this key securely on your local machine.

### Starting a task
Create a project workspace by clicking the New Task button. You assign a specific AI agent to this workspace. The app then opens a terminal window linked to your project folders. 

## 📂 Working with git

Rig simplifies version control. You see a list of every branch in your repository.

* Branch view: Click the Branches tab to see your current progress.
* Switching: Select a branch to make it active. The app handles the checkout process for you.
* Diffs: Click on any file change. Rig displays the original code and the new changes side by side.
* Review: You approve or reject AI suggestions directly from the diff view.

## 🛡️ Privacy and keys

Your data stays on your computer. Rig acts as a middle layer. It sends commands to your AI providers. It sends the files you choose to share. It does not store your source code on our servers. You keep full control over your work.

## ❓ Frequently asked questions

### Does Rig support multiple sessions?
Yes. You run several agents at once. Each session stays in its own tab.

### Where do I find my API key?
You get your API key from your AI provider website. Common providers offer keys through a developer console or dashboard.

### Does the app update itself?
Rig checks for new versions when you start the program. It alerts you if a newer version is available. You choose whether to update at that time.

### Can I use my own terminal settings?
Rig detects your default terminal shell. You change the path to your shell in the customization section if you prefer a different program.

## 🔧 Troubleshooting

### The app does not open
Check if you have an active internet connection. Ensure your antivirus software does not block the installation. Try to run the installer as an administrator.

### The agent returns an error
Verify your API key. Make sure you have current credits with your AI provider. Ensure your network settings allow connections to the agent service.

### The file changes do not appear
Refresh the workspace view. Check that you saved your files in your editor. Ensure you point the workspace to the correct folder.

## 📝 Usage tips
